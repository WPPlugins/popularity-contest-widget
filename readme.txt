The Popularity Contest Widget is a sidebar widget for displaying your blog's top posts as indicated by the Popularity 
Contest plugin.

== Description ==

The Popularity Contest plugin is a great way to highlight the best posts on your blog to your visitors. Unfortunately, 
integrating the Popularity Contest plugin into your blog's theme can be a problem, espectially to none technical users.
The Popularity Contest Widget makes integrating the Popularity Contest plugin into your existing blog theme.

In order for the Popularity Contest Widget to work it *must* have the Popularity Contest plugin already installed and 
activated.

== Installation ==

1. Upload `popularity-contest-widget.php` to the `/wp-content/plugins/` directory
2. In your blog's admin system goto `Presentation > Widgets`
3. Drag ‘n drop the widget to a convenient place in your sidebar
